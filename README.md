# Domain-Specific DevOps Task

Containerized a full-stack application consisting of a **React.js frontend** and a **Go backend**, then prepared it for production deployment.

### Features

* 🐳 Multi-stage Docker builds
* 📦 Docker image optimization
* 🔄 Reverse proxy using Nginx
* ⚖️ Load balancing across multiple backend instances
* 📈 Horizontal scaling
* 🔒 HTTPS with SSL/TLS
* 🛡️ Container and server security hardening
* ❤️ Health checks and automatic restarts
* 🚀 CI/CD pipeline using Jenkins
* 📊 Resource limits for containers
* 🔐 Security headers and rate limiting
* 📧 Email notifications for detected vulnerabilities
* ☁️ Production-ready deployment

**Documentation:**
[DevOPS_Task1/README.md](DevOPS_Task1/README.md)

**Tech Stack**

`Docker` `Docker Compose` `Nginx` `Jenkins` `React` `Go` `PostgreSQL` `CI/CD` `Linux` `Bash` `SSL/TLS`
